KEY FEATURES ILLUSTRATED:

1. HEADER SECTION
   - Milestone name with flag icon
   - Status badge (color-coded)
   - Due date with calendar icon
   - Description text
   - Edit and Delete buttons

2. TAB NAVIGATION
   - Three tabs: Summary, Tasks, Context
   - Tab counts showing number of items
   - Active tab highlighted

3. SUMMARY TAB
   - AI-generated summary display
   - Regenerate button for updating
   - Statistics cards with metrics
   - Clean, readable layout

4. TASKS TAB
   - Table view of all tasks
   - Status and type columns
   - AI-generated indicator
   - Quick view buttons
   - New task creation button

5. CONTEXT TAB (MAIN FEATURE)
   - Add Context Object button with form
   - Tab selection for different types
   - Drag-and-drop file upload area
   - Forms for note/transcript/email input
   
   Context Object Cards:
   - Type icon (emoji) for visual recognition
   - Title and metadata (type, date, user)
   - Analysis status badge
   - Summary display (if analyzed)
   - Derived tasks count
   - Action buttons (Analyze, Create Tasks, Delete)

6. INTERACTIVE ELEMENTS
   - Collapsible forms
   - Drag-and-drop zones
   - Action buttons with icons
   - Status badges
   - Loading states (not shown in ASCII)

7. VISUAL DESIGN
   - Card-based layout
   - Proper spacing and hierarchy
   - Color-coded badges
   - Icon usage for clarity
   - Responsive design principles

## Color Scheme Reference

```
Status Badges:
- Planned:     Secondary (Gray)
- In Progress: Primary (Blue)
- Completed:   Success (Green)

Task Status:
- New:     Secondary (Gray)
- Working: Primary (Blue)
- Review:  Warning (Yellow)
- Ready:   Info (Cyan)
- Done:    Success (Green)

Action Buttons:
- AI Operations: Success (Green) with robot icon
- Edit:          Primary (Blue)
- Delete:        Danger (Red)
- Create:        Primary (Blue)

Context Types:
- File:       Info (Blue)
- Email:      Warning (Yellow)
- Transcript: Secondary (Purple)
- Note:       Success (Green)
```

## User Interaction Flow

```
START
  │
  ├─> Navigate to Item Detail
  │     │
  │     └─> Click on Milestone in list
  │           │
  │           └─> Opens Milestone Detail Page
  │                 │
  │                 ├─> [Summary Tab]
  │                 │     ├─> View AI summary
  │                 │     └─> Click "Regenerate" if needed
  │                 │
  │                 ├─> [Tasks Tab]
  │                 │     ├─> View all tasks
  │                 │     ├─> Click task to view details
  │                 │     └─> Click "New Task" to create
  │                 │
  │                 └─> [Context Tab]
  │                       │
  │                       ├─> Click "Add Context Object"
  │                       │     │
  │                       │     ├─> [File Upload]
  │                       │     │     ├─> Drag & drop files
  │                       │     │     └─> Auto-analyze on upload
  │                       │     │
  │                       │     ├─> [Note Input]
  │                       │     │     ├─> Enter title & content
  │                       │     │     └─> Submit
  │                       │     │
  │                       │     ├─> [Transcript Input]
  │                       │     │     ├─> Enter meeting name & transcript
  │                       │     │     └─> Submit
  │                       │     │
  │                       │     └─> [Email Input]
  │                       │           ├─> Enter subject & content
  │                       │           └─> Submit
  │                       │
  │                       └─> View Context Objects
  │                             │
  │                             ├─> [If Not Analyzed]
  │                             │     └─> Click "Analyze"
  │                             │           └─> AI generates summary & tasks
  │                             │
  │                             ├─> [If Analyzed]
  │                             │     └─> Click "Create Tasks"
  │                             │           └─> Tasks appear in Tasks tab
  │                             │
  │                             └─> Click "Delete" to remove
  │
END
```
