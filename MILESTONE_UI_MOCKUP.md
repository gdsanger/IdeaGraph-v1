## Color Scheme Reference

```
Status Badges:
- Planned:     Secondary (Gray)
- In Progress: Primary (Blue)
- Completed:   Success (Green)

Task Status:
- New:     Secondary (Gray)
- Working: Primary (Blue)
- Review:  Warning (Yellow)
- Ready:   Info (Cyan)
- Done:    Success (Green)

Action Buttons:
- AI Operations: Success (Green) with robot icon
- Edit:          Primary (Blue)
- Delete:        Danger (Red)
- Create:        Primary (Blue)

Context Types:
- File:       Info (Blue)
- Email:      Warning (Yellow)
- Transcript: Secondary (Purple)
- Note:       Success (Green)
```
